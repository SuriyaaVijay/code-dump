## Run Locally

Clone the project

```bash
  git clone https://github.com/tusharkhatri434/my-youtube.git
  
  Add your Google API key : IN constant.js file.
```

Go to the project directory

```bash
  cd Food-villa
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
